# Nothing Design System — Philosophy

- **Subtract.** Elements must earn pixels. Default: removal.
- **Structure as ornament.** Expose grid, data, hierarchy.
- **Monochrome canvas.** Color is an event, not default (except data status).
- **Typographic hierarchy.** Scale, weight, spacing drive hierarchy — not color/icons/borders.
- **Dual-mode parity.** Dark: OLED black. Light: warm off-white. Both first-class.
- **Industrial warmth.** Precise but human.
