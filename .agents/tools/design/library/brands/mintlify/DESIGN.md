# Design System: Mintlify

White, airy, documentation-as-product aesthetic. Brand accent: `#18E299` (green). Fonts: Inter (all text) + Geist Mono (code/technical). Signature shape: full-pill (9999px) buttons and inputs.

## Quick Reference

**Colors:** Background `#ffffff` · Text `#0d0d0d` · Brand `#18E299` · Border `rgba(0,0,0,0.05)`

**Typography:** Inter 400/500/600 · Geist Mono for code · Display: 64px/-1.28px · Body: 16–18px/normal

**Radius:** 4px tags · 8px nav · 16px cards · 24px featured · 9999px buttons/inputs/pills

**Spacing:** 8px base · 24–32px card padding · 48–96px section padding
