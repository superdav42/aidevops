# Do's and Don'ts

## Do's

- Use `#76b900` as accent only — borders, underlines, and highlights
- Use transparent buttons with green borders by default
- Apply weight 700 for all interactive and heading elements
- Use 2px border radius for everything (buttons, cards, containers, inputs)
- Use white text on dark sections; black text on light sections
- Apply `#3860be` (blue) for all link hover states regardless of default color
- Use line-height 1.25 for headings, 1.50-1.67 for body text
- Use uppercase 14px bold for navigation links

## Don'ts

- Don't use `#76b900` as a background fill — it's a signal color only
- Don't use filled button backgrounds by default — only on hover/active states
- Don't use weight 400 for interactive elements — reserve for body paragraphs only
- Don't use border radius > 2px (except 50% for circular avatars)
- Don't mix link hover colors — always `#3860be` regardless of context
- Don't use glassmorphism or blur effects — clarity over atmosphere
- Don't use decorative letter-spacing — only compact buttons use 0.144px
