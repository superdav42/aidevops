# Design System: Agency Techie

Dark-mode-first design system for developer tooling and technical products. Terminal-inspired aesthetic with near-black surfaces, bright cyan primary (`#22d3ee`), monospace accents, and sharp geometry.

## Quick Reference

**Colours:** `#22d3ee` primary · `#a78bfa` accent · `#e2e8f0` text · `#0d1117` bg · `#161b22` surface-1

**Type:** JetBrains Mono (code/data) · Inter (prose/UI) · Body 15px/400 · H1 36px/700

**Radius:** 4px default · 6px cards · 2px badges

**Motion:** 150ms ease-out — state transitions only
